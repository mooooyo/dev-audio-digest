# 2026-05-22 frontend 뉴스 정리
프론트엔드 개발자 관점에서 지난주에 일어난 주요 뉴스를 요약합니다.

## Next.js
Next.js v16.3.0-canary.26이 출시되었습니다. 주요 변경점으로는 다음과 같습니다.
- `TURBO_TASKS_AVAILABLE_PARALLELISM` 환경 변수가 추가되었습니다.
- `--debug-build-paths` 플래그가 `src/` 디렉토리 프로젝트에서도 작동합니다.
- Linux Playwright 작업이 Microsoft 컨테이너에서 돌아갑니다.
- App Shell 프리페치를 서버에서 처리합니다.
-TurboPack에서 ctor을 비활성화했습니다.

자세한 정보는 [Next.js 깃허브](https://github.com/vercel/next.js/releases/tag/v16.3.0-canary.26)를 참고하세요.
