# 2026-03-27 frontend 뉴스 정리
오늘은 다양한 프론트엔드 관련 뉴스가 있습니다. 크롬에서 새로운 뷰 전환 기능이 추가되었으며, 다양한 오픈소스 프로젝트들이 공개되었습니다.

## 크롬 뷰 전환 기능
크롬에서 새로운 뷰 전환 기능이 추가되었습니다. 이 기능은 [엘리먼트 범위 뷰 전환](https://developer.chrome.com/blog/element-scoped-view-transitions?hl=en)을 허용하여 웹 개발자가 더 나은 사용자 경험을 제공할 수 있게 합니다.

```javascript
//엘리먼트 범위 뷰 전환 예제
element scoped view transition
```

詳細한 정보는 [크롬 개발자 블로그](https://developer.chrome.com/blog/element-scoped-view-transitions?hl=en)를 참조하세요.

## 오픈소스 프로젝트
다양한 오픈소스 프로젝트들이 공개되었습니다.

* [figma/mcp-server-guide](https://github.com/figma/mcp-server-guide): 피그마 MCP 서버 가이드
* [breschio/drawbridge](https://github.com/breschio/drawbridge): 크로메 확장 프로그램으로 클로드 코드와 커서에 피그마 스타일의 注釈를 추가하는 도구
* [atlassian/atlassian-mcp-server](https://github.com/atlassian/atlassian-mcp-server): 아틀라시안 MCP 서버
* [jgraph/drawio](https://github.com/jgraph/drawio): 다이어그램 편집기
* [FortAwesome/Font-Awesome](https://github.com/FortAwesome/Font-Awesome):.Font Awesome
* [usebruno/bruno](https://github.com/usebruno/bruno): 오픈 소스 API 클라이언트

이 외에도 많은 프로젝트들이 있습니다. 자세한 정보는 각 프로젝트의 깃허브 페이지를 참조하세요.
