# 2026-04-12 frontend 뉴스 정리

## Next.js
프론트엔드 개발자로 오늘 가장 주목할べき 뉴스는 **Next.js**입니다. vercel/next.js의 v16.2.1-canary.33 버전이 출시되었습니다. [원문](https://github.com/vercel/next.js/releases/tag/v16.2.1-canary.33)

```bash
npm install next@latest
```

이번 릴리즈에는 몇 가지 **breaking change가 있습니다**. 개발자들은 이 부분에 주의해야 합니다. 예를 들어, `turbopack`에 LocalPathOrProjectPath PostCSS config resolution이 추가되었습니다.
