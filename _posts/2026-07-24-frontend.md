# 2026-07-24 frontend 뉴스 정리
## Next.js
Next.js v16.3.0-canary.95이 출시되었습니다. [원문](https://github.com/vercel/next.js/releases/tag/v16.3.0-canary.95)

### 변경 사항
- Turbopack: Very minor improvements for watcher loop: [#96103](https://github.com/vercel/next.js/pull/96103)
- Turbopack: Refactor watcher event handling and batching logic: [#96087](https://github.com/vercel/next.js/pull/96087)
- `url.parse()` 대신 `new URL()`를 사용하여 커스텀 서버 예제에서 deprecated 함수를 제거했습니다: [#96105](https://github.com/vercel/next.js/pull/96105)
- React를 `711c445b-20260722`에서 `28cd4bb0-20260723`로 업그레이드했습니다: [#96100](https://github.com/vercel/next.js/pull/96100)
- `generateStaticParams` 결과가 비어 있거나 불완전한 경우 예외를 발생시킵니다: [#95969](https://github.com/vercel/next.js/pull/95969)
- `generateStaticParams`에서 유효하지 않은 값을 반환할 경우 예외를 발생시킵니다: [#95968](https://github.com/vercel/next.js/pull/95968)
- 실험적 인 `viewTransition` 플래그를 제거했습니다: [#96098](https://github.com/vercel/next.js/pull/96098)
