# 2026-04-13 frontend 뉴스 정리

## NEXT.js
 NEXT.js의 새로운 버전인 `v16.2.1-canary.36`과 `v16.2.1-canary.35`이 출시되었습니다.  
이 버전에서는 주요 변경 사항으로는 prefectch hints를 처리하는 방식의 변경이 있습니다. 
```diff
- Fix prefetch hints not being loaded during revalidation: #92625
```
또한, `v16.2.1-canary.35` 버전에서는 Compiler에서 boolean과 number primitive를 Next.js의 config define에서 지원하도록 업데이트되었습니다. 
```diff
- Compiler: Support boolean and number primtives in next.config defines: #92731
```
그외에도 Turbopack과 관련된 여러 변경 사항이 있습니다. 
```diff
- Turbopack: properly remove unused TS enums with scope hoisting: #88205
- Turbopack: Add import.meta.glob support (Vite compat): #92640
- turbo-tasks: Fix recomputation loop by allowing cell cleanup on error during recomputation: #92725
```
 NEXT.js의 전체 변경 사항은 [NEXT.js 깃허브](https://github.com/vercel/next.js/releases/tag/v16.2.1-canary.36)에서 확인할 수 있습니다.
