# 2026-07-13 frontend 뉴스 정리
## NEXT.js
NEXT.js는 버전 `v16.3.0-preview.6`을 출시했습니다. 이 버전에서는 다음과 같은 변경 사항이 있습니다:
- ESLint 규칙 `no-location-assign-relative-destination` 개선
- Turbopack에서 CSS 모듈을 chunk-group co-occurrence에 따라 선형으로 정렬
- 테스트 코드와 관련된 다양한 변경 사항
