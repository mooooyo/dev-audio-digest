# 2026-06-01 frontend 뉴스 정리
지난주에_compare한 프론트엔드 개발자 관점에서의 주요 뉴스를 정리하였습니다.

## 라이브러리 및 프레임워크
### React
* React 19.2.7, 19.1.8, 19.0.7 버전이 출시되었습니다. Server Components에서 `FormData` 항목이 누락되는 버그가 수정되었습니다.
* [React Releases](https://github.com/facebook/react/releases)

### Next.js
* Next.js 16.2.7 버전이 출시되었습니다. 문서 고침 및 여러 버그修复 항목이 포함되어 있습니다.
* [Next.js Releases](https://github.com/vercel/next.js/releases)
